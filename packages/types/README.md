# @elinzy/e-types

A package of types for use in e, or projects wishing to extend them.

## Install

```shell
npm install -D @elinzy/e-types
```

## Usage

```ts
import type { EOptions } from '@elinzy/e-types';

function example(opts: EOptions) {
    // do nothing
}
```
