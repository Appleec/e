# divide

Divide two numbers.

## 版本

0.4.11

## 使用

- 语法

```js
divide(dividend: number, divisor: number): number
```

- 参数

| Name          | Type        | Required | Description                       |
|---------------|-------------|----------|-----------------------------------|
| dividend      | number      | Y        | The first number in a division.   |
| divisor       | number      | Y        | The second number in a division.  |

- 返回值

Returns the quotient.

## 示例

```js
divide(6, 4)
// => 1.5
```
