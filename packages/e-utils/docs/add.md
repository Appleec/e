# add

Adds two numbers.

## 版本

0.4.11

## 使用

- 语法

```js
add(augend: number, addend: number): number
```

- 参数

| Name        | Type        | Required | Description                       |
|-------------|-------------|----------|-----------------------------------|
| augend      | number      | Y        | The first number in an addition.  |
| addend      | number      | Y        | The second number in an addition. |

- 返回值

Returns the total.

## 示例

```js
add(6, 4)
// => 10

add(-1, 4)
// => 3
```
