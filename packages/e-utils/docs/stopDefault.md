# stopDefault

Block the browser's default behavior

## 版本

0.2.0

## 使用

- 语法

```js
stopDefault(e) -> {void}
```

- 参数

| Name   | Type          | Description    |
|--------|---------------|----------------|
| e      | Event         |                |

- 返回值

No return value.

## 示例

```js
stopDefault();
// => void
```
